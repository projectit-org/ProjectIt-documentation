Styling the Web Application and the Editor

The web application, including the editor, can be styled using the scss files in the folder ~webapp/style. There are two sets of CSS variables that determine the look of the web application, one for the light theme, and one for the dark theme. The latter can be found in the folder ~webapp/style/dark.

The comments and naming should (hopefully) indicate where each variable is being used.

// docu-project/webapp/style/_freon_theme-light.scss

@use 'sass:color';

@use '@material/theme/color-palette';

// Freon Colors!!
@use '@material/theme/index' as theme with (
  //$primary: #80cbc4,
  $primary: #4f9a94,
  //$secondary: #ce93d8,
  $secondary: #ffcc80,
  $surface: #eeeeee,
  $background: #fff,
  $error: #B00020,
  //$on-primary: color-palette.$orange-50,
  $on-secondary: #00695c,   // a slighter darker version of 'primary'
  //$on-surface: black,
  //$on-error: black,
  //accessible-ink-color,
  //contrast,
  //contrast-tone,
  //luminance,
  //text-emphasis,
  //tone
);

// Include all the global styles that do not involve dark/light modes.
